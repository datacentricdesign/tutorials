# DCD Tutorials

Hands-on tutorials for the **Data-Centred Design for Connected Products** course, built around the [Data Foundry](https://data-foundry.net) platform.

Each tutorial is tied to a specific chapter and section of the textbook. They are designed to be completed in order within a chapter, but most can also stand alone.

---

## Structure

| Chapter | Topic | Tutorials |
|---------|-------|-----------|
| [Chapter 3 — Data Collection](ch3-collection) | APIs, data donation, sensors, ESM | T3.1 – T3.4 |
| [Chapter 4 — Data Exploration](ch4-exploration) | Time series, visual, text | T4.1 – T4.3 |
| [Chapter 5 — Data Analytics](ch5-analytics) | Statistical, visual, text analysis | T5.1 – T5.3 |
| [Chapter 7 — Realisation](ch7-realisation) | Sensors, pipelines, interfaces, LLMs | T7.1 – T7.7 |

---

## How to use these tutorials

Each tutorial page includes:

- **Textbook context** — the section of the DCD book this tutorial accompanies
- **Data Foundry features** — the platform capabilities you will use
- **Platform pages** — direct links to relevant Data Foundry documentation
- **Steps** — a numbered procedure to follow
