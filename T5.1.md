# T5.1 · Time-Series Data Analysis

**Location:** `textbook/5/analytics.tex:133–136`
**Textbook context:** Aggregation, summary statistics, trend analysis, correlation, PDA.
**Data Foundry features:** IoT Dataset, Starboard Notebook, Python client
**Platform pages:** [Starboard Notebook](https://data-foundry.net/Guides/Tools/Starboard/StarboardNotebook.html) · [Python client](https://data-foundry.net/Guides/Clients/Python.html)

## Steps

1. **Load cleaned dataset** – Pull the IoT Dataset from T3.3 / T4.1 into a Starboard Notebook via the Python client.
2. **Summary statistics** – Compute mean, median, std, IQR per day and per participant. Identify high-variance participants vs. stable ones.
3. **Trend analysis** – Fit a linear regression or compute a rolling mean over the study period. Is there a directional change? Could it be explained by an intervention or external event?
4. **Correlation** – Compute pairwise correlation between two sensor channels (e.g. temperature and motion). Plot a scatter matrix. Formulate a design hypothesis.
5. **PDA visualisation** – Produce a clean, labelled figure (e.g. radial daily plot or annotated time series) for use as a PDA stimulus. Describe 2–3 questions you would ask a participant looking at it.
6. **Reflect** – Which technique revealed the most design-relevant insight? What would you add to the collection protocol if you were to run this study again?
