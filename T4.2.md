# T4.2 · Exploring Visual Data

**Location:** `textbook/4/exploration.tex:355–358`
**Textbook context:** Video logging, thematic compilation, anonymisation, preprocessing for CV.
**Data Foundry features:** Media Dataset, Annotation Dataset
**Platform pages:** [Media Dataset](https://data-foundry.net/Learning/Datasets/DatasetsOverview/mediadataset.html) · [Annotation Dataset](https://data-foundry.net/Learning/Datasets/DatasetsOverview/annotationdataset.html)

## Steps

1. **Upload footage** – Create a Media Dataset in Data Foundry; upload a short set of images or video frames (from a field session or provided sample).
2. **Anonymise first** – Before any analysis, review frames for faces, licence plates, and identifying details. Apply face-blurring (e.g. via a local Python script using OpenCV); re-upload anonymised versions.
3. **Video log** – Browse the Media Dataset and create a timestamped log of notable moments in an Annotation Dataset linked to the media. Use brief descriptive notes, not interpretations yet.
4. **Thematic compilation** – Group annotations by emerging themes. Select 3–5 clips or frame sequences that illustrate a candidate pattern.
5. **Reflect on positionality** – Review your log: which moments did you mark? What did you pass over? What does that reveal about your analytical lens?
6. **Prepare for collaborative analysis** – Export the thematic clips for a Video Card Game session (see Chapter 5).
