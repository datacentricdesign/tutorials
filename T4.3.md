# T4.3 · Exploring Text Data

**Location:** `textbook/4/exploration.tex:481–484`
**Textbook context:** RTA phases 1–3; transcription, anonymisation, familiarisation, initial coding.
**Data Foundry features:** Form Dataset or Diary Dataset, Annotation Dataset, Transcription tool
**Platform pages:** [Form Dataset](https://data-foundry.net/Learning/Datasets/DatasetsOverview/formdataset.html) · [Diary Dataset](https://data-foundry.net/Learning/Datasets/DatasetsOverview/diarydataset.html) · [Transcription tool](https://data-foundry.net/Guides/Tools/Transcription.html)

## Steps

1. **Collect or load text** – Either use responses from a Form or Diary Dataset collected earlier, or upload an interview transcript to an Existing Dataset.
2. **Transcribe** – If working from audio, use the Data Foundry Transcription tool (local Whisper model) to produce a verbatim transcript. Review the output for errors.
3. **Anonymise** – Work through the transcript substituting participant names, places, and identifying details with pseudonyms. Document changes in a separate key file (stored separately from the transcript).
4. **Familiarise (RTA Phase 1)** – Read the full corpus without coding. Write a 200-word reflexivity note: what surprised you? What felt familiar? What did you skim past?
5. **Generate initial codes (RTA Phase 2)** – Re-read and label segments with short codes in an Annotation Dataset linked to the text corpus.
6. **Candidate themes (RTA Phase 3)** – Group codes into 3–5 candidate themes; write a one-sentence definition of each. Note which themes would need more data to support.
